namaste friends.

  

There is a new app ( <https://play.google.com/store/apps/details?id=sanskritcode.sanskritdictionaryupdater>)
to update your stardict-compatible stardict dictionaries without having to manually download and extract each one.  
  
Code for this is available for any one to copy and modify [here](https://github.com/vvasuki/sanskritnlpjava/tree/master/src/main/android-apps). (May a 1000 apps bloom!) If you want to contribute new dictionaries to the [stardict-sanskrit project](https://github.com/vvasuki/stardict-sanskrit), that is welcome too.

  

Details in English:

This program downloads latest dictionary files from the internet, and stores it in the SDCARD/dictdata folder, for Stardict apps to use. We assume that you already have or will install Stardict-compatible dictionary utilities (Eg: ColorDict, GoldenDict ...) separately.

We use the index maintained by the stardict-sanskrit project (See:[https://github.com/vvasuki/stardict-sanskrit](https://www.google.com/url?q=https://www.google.com/url?q%3Dhttps://github.com/vvasuki/stardict-sanskrit%26sa%3DD%26usg%3DAFQjCNFShDEMH6M0ESz0_zHA4eUeG7_UTA&sa=D&usg=AFQjCNFPBPu67asiNdOcEtYqsLfkmF-BBA)[https://sites.google.com/site/sanskritcode/](https://www.google.com/url?q=https://www.google.com/url?q%3Dhttps://sites.google.com/site/sanskritcode/%26sa%3DD%26usg%3DAFQjCNEN_XxlLFcVTUCW0mxWENGwws7Wcg&sa=D&usg=AFQjCNFtzsxYjs0-qyQOCATsR-TsWM7XZg)).

  

संस्कृतेन विवरणम् -

प्राप्यान्तर्जालात् संस्कृत-स्टार्डिक्ट-कोशानां सूचिं, स्थापयति तान् कोशान् यन्त्रे SDCARD/dictdata इत्यत्र। एवं स्थापिताः सञ्चिकाः stardict-प्रयोक्तृ-तन्त्रांशैः क्रियतामिति। स्टार्डिक्ट्-प्रयोक्तृ-तान्त्रांशः कश्चित् (उदाहरणार्थम् - ColorDict, GoldenDict ...) युष्माभिः स्थापितो वा स्थाप्यते वेति नः प्रत्ययः।

  

वयं stardict-sanskrit-गणेन प्रकाशितां कोशसूचिं प्रयुञ्ज्महे। (ईक्षन्ताम् : <https://github.com/vvasuki/stardict-sanskrit> <https://sites.google.com/site/sanskritcode/> )
