**संस्कृतम्(फ़ोनेटिक्) इत्यनेन** देवनागरीटङ्कने ITRANS च Harvard-Kyotoइत्येतयोः (ययोः विवराणि लभ्यन्ते [अत्र](http://en.wikipedia.org/wiki/Devanagari_transliteration)) किञ्चन सम्मिश्रणम् कृतम् अस्ति। तदतिरिच्य ट्ङ्कनसौलभ्याय अन्ये इमे नियमाः योजिताः -  
R -> ऋ, RR -> ॠ  
nk -> ङ्क्, nkh -> ङ्ख्, nch/ nc -> ञ्च्, nT -> ण्ट् इत्यादयः,  
.a -> ऽ, .N -> ँ  
\|\| वा .,->॥, \| वा .. -> ।, ... -> …, 'rAma. ' -> 'राम। ' (अत्र रिक्तस्थानयुक्तम् बिन्दुटङ्कनेन दण्डस्य उत्पत्तिः। )
