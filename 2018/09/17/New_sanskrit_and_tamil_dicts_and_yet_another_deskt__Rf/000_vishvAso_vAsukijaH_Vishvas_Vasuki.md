There is a new way of installing dicts on your desktop computer. It is number 1 below:

-   Install Java and
    use[dict-tools](https://github.com/sanskrit-coders/dict-tools)jar:
    -   Download[dict-tools.jar](https://github.com/sanskrit-coders/dict-tools/raw/master/bin/artifacts/dict-tools.jar).
    -   `java -jar bin/artifacts/dict-tools.jar install /home/someGuy/sanskrit-coders/stardict-dicts-installed/https://raw.githubusercontent.com/sanskrit-coders/stardict-dictionary-updater/master/dictionaryIndices.md` -   Install python 2.7 and use <https://github.com/nangia/pydictupdater>

  

Why did I create it? I'd forgotten all about +sandeep's [pydictupdater](https://github.com/nangia/pydictupdater). Well, anyway -at least it was a scala refresher and it uses more up-to-date indices.  

  

PS: other dict installation tips are at <https://sanskrit-coders.github.io/site/pages/dictionaries/offline.html#how-to-install-and-use-dictionaries-on-your-device>
