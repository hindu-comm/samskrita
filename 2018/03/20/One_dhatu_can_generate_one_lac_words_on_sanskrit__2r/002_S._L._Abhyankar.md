निम्नं विवृतान्यपि विचारणीयानि

1.  अष्टाध्याय्यां "प्रत्ययः"-इत्यस्य अधिकार-सूत्रस्य अधिकारः
    तृतीय-चतुर्थ-पञ्चम-इत्यादिषु त्रिषु अध्यायेषु वर्तते / तत्र बहवः
    बहवः प्रत्ययाः / तत्रापि "उणादयः बहुलम्" इत्यनेन सूत्रेण बहूनां
    बहूनां प्रत्ययानां निर्देशः / 2.  धातुभ्यः उपसर्गाः अपि भवन्ति / उपसर्गाणामनेके पर्यायाः अपि यथा
    अध्याय-शब्दे अधि+आ+यः एवं द्वौ उपसर्गौ सार्धं योजयितुं शक्यम् /  
3.  सन्-णिच्-आदयः द्वादश प्रयोगाः भवन्ति / तेषामपि प्रयोगान्तराण्यपि
    भवन्ति यथा कृ => करोति => क्रियते (कर्मणिप्रयोगे); कृ = कारयति/ते
    (णिचि) => कार्यते (णिचः कर्मणिप्रयोगे)
4.  शब्दानां विविधानि तद्धितान्यपि भवन्ति / तेषामपि सुबन्तानि भवन्ति /
    तद्धितानि विशेषणानि चेत् सुबन्तानि त्रिष्वपि लिङ्गेषु / 5.  इतो बहूनि विवरणान्यपि सम्भवनीयानि /
