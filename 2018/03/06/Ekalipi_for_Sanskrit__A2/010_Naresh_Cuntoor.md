Script-related threads seem to appear with exhausting regularity. As has been stated by others in previous threads, Samskrit is written using several scripts of which Devanagari is probably the most popular.  
  
Discussion about fonts may be useful. But other than that, these 'my script is better than yours' threads serve little practical purpose IMO. As far as the group goes, please use Devanagari or English transliteration.
