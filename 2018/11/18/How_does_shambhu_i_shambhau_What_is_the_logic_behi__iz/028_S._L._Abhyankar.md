यथा मया अवगतम् =>

शंभु-शब्दस्य सप्तम्येकवचनम् प्राप्तुं प्रक्रिया प्रायः निम्नमिव =>

प्रातिपदिकम् शंभु प्रत्ययःङि

ङि-प्रत्ययः इ-इत्येव योजनीयः यतः अत्रङ्-कारः इत् अस्ति लशक्वतद्धिते (1'3'8) इत्यनेन सूत्रेण \|

"शंभु"इत्यस्मिन् प्रातिपदिके अन्त्यः उकारः (शम्भ्+उ) मननीयः\|

इदुद्भ्याम् (7'3'117) औत् (7'3'118) अच्च घेः (7'3'119) एतैः सूत्रैः =>

(शम्भ्+ उ) + (इ) => (शम्भ्+ अ) + (औ) = शम्भौ \|
