Valmiki Ramayanam Uttarakanda sarga 16 sloka 36 to 39

  

samvatsara sahasram tu rudato rakshaso gatam /

thathah prito mahadevah sailagre vishthitah prabhuh //

  

muktvaa chaasya bhujaan rama praaha vaakyam dasaananam /

pritosmi tava viiryasya saundiiryaat cha dasaanana //

  

sailaakraantena ye muktah tvayaa raavah sudaarunah /

yasmaat lokatrayam chaidat raavitam bhayamaagatam //

  

tasaatvam raavanonaama naamnaa raajan bhavishyati /
