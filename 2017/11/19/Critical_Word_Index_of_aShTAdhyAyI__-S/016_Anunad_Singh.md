नमस्ते साधुवादश्च अभ्यंकरकुलोत्पन्न-श्रीपाद-महोदय!  
  
वर्तमान विषयस्य विविधपक्षाणि विचिन्त्य एतत्शब्दसंकलनसमस्यायाः हलविषये  
मम इदं मतम्-  
  
0) Supposing that a text file of the book is available (in appropriate form).  
  
1) This critical word index can best be compiled by a combination of  
manual interaction and a simple software tool.  
  
2) The user having sufficient knowledge of Sanskrit to identify  
पदानि/शब्दानि , will input a शब्दम् and corresponding पदानि.  
  
3) The software will then search those पदानि in the text file (of  
ashTAdhyAyI, for example). For this it will also be able to do  
सन्धि/समास विच्छेद. The software will write all the occurrences of  
that शब्दम् in a seperate file. The software will make another file in  
which all those पदानि will be shown striked out so that the user will  
find it easy to specify a new शब्दम् and corresponding पदानि for new  
search.  
  
4) By repeating the above steps, this 'critical word list' can be made.  
  
5) In addition to above, the software should be able to take as input  
a text file containing some of शब्दम् and corresponding पदानि (in a  
simple format) too and search those words.
