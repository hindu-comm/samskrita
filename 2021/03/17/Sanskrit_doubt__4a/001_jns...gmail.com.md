III i) वयं भारतीयाः - कर्तृ पदं = वयम् , कर्तृपदस्य क्रियापदम् = अग्रेसरामः \[कर्तृ पदं = subject/doer, क्रियापदं = verb, अग्रेसरामः = march ahead\]

ii\) परिचित - विपरीत पदं = अपरिचित \[परिचित = familiar, अपरिचित = stranger\]

iii\) एतत् सर्वं एकेन "एव" यन्त्रेण कर्तुं शक्यते \[एव is अव्ययम् = alone/only/itself, all this can be done by one machine alone/only/itself\]

iv\) प्रदानाय => पर्याय पदं = प्रदातुम् \[=to liberally donate\]
