Ramayana - aranya Kanda

srga 18 - slokas 26 to 30.

Surpanakaha meets Rama and asks him

to discard Sita and marry her.

  

aham prabhavasampannaa

svacchandabalagaaaminii /

chiraaraya bhava me bhartaa

sitayaa kim karishyasi //

  

vikruta cha viruupaa cha

na cheyam sadrusiii tava/

ahamevaanurupaa te

bhaaryaarupena pasya maam //

  

imaam viruupaam asatiim

karaalaam nirnatodariim //

  

  

ramayana -aranyakanda sarga 34

  

surpanakha tells ravana about sita -

  

raamasya tu visaalaakshii

purnendu-sadrusaananaa /

dharmapatni priyaa bhartuh

nityam priyahite rataa //

  

saa sukesii sunaasoruu

saruupaa cha yasasvinii /

devateva vanasyaasya

raajate sriirivaaparaa //

  

taptakaachana-varnaabhaa

raktatunganakhii subhaa /

siitaa naama varaarohaa

vaidehii tanumadhyamaa //

  

naishaa devii na gandharvii

na yakshii na cha kinnarii /

naivam ruupaa mayaa naarii

drishtapuurvaa mahiitale //

  

yasya siitaa bhavet bhaaryaa

yam cha hristhaa parishvajet /

atijivetsa sarvesham

lokeshvapi puramdaraat //

  

saa susiilaa vapuslaakhyaa

ruupenaapriyatamaa bhuvi /

tavaanurupaa bharyaa syaat

tvam cha thasyaah thathaa patih//

  

taam tu vistirnajaghanaam

piinasroni-payodharaam /

bhaaryaarthe cha tavaanetum

udyataaham varaananaam //

  

virupitaasmi krurena

lakshmanena mahaabhuja /

taam drishtvaadya cha vaidehiim

purnachandranibhaananaam /

manmathasya saraan vai

tvam vidheyo bhavishyasi /

yadi tasyaamabhipraayo

bhaaryaarthe tava jaayate //

  

sighramuddhriyataam paado

jaayaarthamiha dakshinah /

kuru priyam thathaa theshaam

rakshasaam raakshaseswara //

  

vadhat thasya nrusumsasya

raamasyaasaramavaasinah /

tam sarairnihatam hatvaa

lakshmanam cha mahaaratham //

  

hatanaathaam sukham siitaa

yathaavad upabhokshyasi /

rochate yadi te vaakyam

mamaitad raakshasesvra //

  

kriyataam nirvisankena

vachanam mama raavana /

vijnaayehaatmasaktim cha

hriyataamabalaa ballat //

  

siitaa sarvaanavandyaangii

bhaaryaarthe raakshasesvra /
