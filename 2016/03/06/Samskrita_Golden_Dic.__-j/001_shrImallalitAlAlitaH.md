1\. Open a notepad.  
2. Paste it:  

> > body >
> > { >
> >  margin-top: 1px; >
> >  margin-right: 3px; >
> >  margin-left: 2px; >
> >  margin-bottom: 3px; >
> >  background: white; >
> >  font-family: Siddhanta; >
> >  font-size:18px; >
> > } >
> >   

  

  

3. Go to File->Save as

4\. Type this in the name field - 
%appdata%\\GoldenDict\\article-style.css

5\. Press Enter.

6\. Relaunch GoldenDict (If it was open then exit it, then open; mere closing will not work)

  

This was for windows.

  

For linux you have to put it at *\~/.goldendict*

  

If you are interested, please go through this page :
<http://goldendict.org/wiki/index.php/FAQ#How_do_I_change_fonts_for_only_a_specific_language.3F>
