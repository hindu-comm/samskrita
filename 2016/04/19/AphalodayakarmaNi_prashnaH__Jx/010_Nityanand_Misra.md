Dr. Bhat is correct. And even if one were to use टाप् from आफलोदयकर्मन् by hook or crook, the resulting lemma (base form) would be आफलोदयकर्मणा and not आफलोदयकर्मा which one would get with डाप्. The reason being with डाप् there is टिलोप (elision of अन् part of –कर्मन्) due to टेः (पा॰सू॰ ६.४.१४३, SK: डिति परे भस्य टेर्लोपः स्यात्) resulting in –कर्म्+आ = –कर्मा which is then inflected as आफलोदयकर्मा आफलोदयकर्मे आफलोदयकर्माः. There is no such टिलोप with टाप्.

  

To understand this further, refer to the below examples from Pt. Ishvarchandra’s and Somalekha’s commentary on the SK. Note that all the forms on the right below (वाचा, etc) are also the forms of the singular instrumental case of the word on the left (वाच्, etc) as except for the अनुबन्ध, there is no difference between टाप् and टा from the perspective of प्रक्रिया (there is a difference in विवक्षा though).



वाच् + टाप् = वाच् + आ = वाचा

निश् + टाप् = निश् + आ = निशा

दिश् + टाप् = दिश् + आ = दिशा

क्षुध् + टाप् = क्षुध् + आ = क्षुधा

गिर् + टाप् = गिर् + आ = गिरा

तृष् + टाप् = तृष् + आ = तृषा

तृष् + टाप् = तृष् + आ = तृषा

रुज् + टाप् = रुज् + आ = रुजा

आपद् + टाप् = आपद् + आ = आपदा

प्रतिपद् + टाप् = प्रतिपद् + आ = प्रतिपदा

  

Similarly, कर्मन् + टाप्, if it would ever happen (it does not), would result in कर्मणा and not कर्मा.
