DearSir,

  

Rule 1: (Savarna Deerga Sandhi) If any *simple* vowel (short or long) is followed by a similar simple vowel (short or long), the two vowels blend into one long similar vowel (Pāṇ. VI.I, 101);

अ + अ = आ (1.1)

अ + आ = आ (1.2)

आ + अ = आ (1.3)

आ + आ = आ (1.4)

  

Rule 2: (Guna Sandhi) If अ *a* or आ *ā*, followed by the dissimilar vowels इ *i*, उ *u*, ऋ *ṛ* (short or long), blends with इ *i* or ई *ī* into the गुण Guṇa ए *e*; with उ *u* or ऊ *ū* into the गुण Guṇa ओ *o*^(\*) ; with ऋ *ṛ* or ॠ *r̥̄* into the गुण Guṇa अर् *ar* (Pāṇ. VI.I, 87);

अ + इ = ए (2.1)

आ + इ = ए (2.2)

अ + उ = ओ (2.3)

आ + उ = ओ (2.4)



Rule 3: (Vriddhi Sandhi) अ *a* or आ *ā*, followed by the diphthongs ए *e*, ओ *o*, ऐ *ai*, or औ *au*, blends with ए *e* into the वृद्धि Vṛddhi ऐ *ai*; with ऐ *ai* also into ऐ *ai*; with ओ *o* into the वृद्धि Vriddhi औ *au*; with औ *au* also into औ *au* (Pāṇ.VI.1,88); e.g.

अ + ए = ऐ (3.1)

आ + ए = ऐ (3.2)

अ + ओ = औ (3.3)

आ + ओ = औ (3.4)



Let us apply the above rules in reverse direction.

ऐ = अ + ए     Vriddhi resolved by (3.1)

 = अ + (अ + इ) Guna resolved by (2.1)

 = (अ + अ) + इ 

 = (आ) + इ by (1.1)

 = ए    by (2.2)

So ऐ *ai* = ए *e*

  

Let us apply similarly to औ

औ = अ + ओ      Vriddhi resolved by (3.3)

 = अ + (अ + उ)    Guna resolved by (2.3)

 = (अ + अ) + उ

 = (आ) + उ by (1.1)

 = ओ        by (2.4)

So औ *au* = ओ *o*



ऐ *ai* = ए *e* and औ *au* = ओ *o*

  

The above mathematical equations giving to **Vriddhi = Guna.**
