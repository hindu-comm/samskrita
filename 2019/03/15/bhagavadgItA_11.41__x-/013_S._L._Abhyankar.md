Sorry that there was a typo in my previous message please read यि in place of य all over.

एचोऽयवायावः simply means

ए changes to अय् Hence सखे इति = सख् **ए** इति = सख् **अय्** इति =सखय् इति = सख**यि**ति

-   OR (by लोपः of य्) सखे इति = सखय् इति = सख इति
-   also in गीता 2'9 योत्स्ये इति = योत्स्य इति

ऐ changes to आय् e.g. नद्यै अर्पणम् = नद् य् **ऐ** अर्पणम् = नद् य् **आय्** अर्पणम् =नद्याय् अर्पणम् =नद्यायर्पणम्

-   OR नद्या (य्) अर्पणम्

ओ changes to अव् e.g. धेनो आगच्छ = धेन् **ओ** आगच्छ = धेन् **अव्** आगच्छ =धेनव् आगच्छ = धेनवागच्छ

-   OR धेन (व्) आगच्छ

औ changes to आव् e.g. पूजार्हौ अरिसूदन (गीता 2'4) = पूजार् ह् **औ** = पूजार् ह् **आव्** अरिसूदन =पूजार्हाव् अरिसूदन = पूजार्हावरिसूदन

-   OR पूजार्हा (व्) अरिसूदन

Hope, this helps.
