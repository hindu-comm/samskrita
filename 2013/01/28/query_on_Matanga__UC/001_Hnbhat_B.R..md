From Todala Tantra:

  

धूमावती महाविद्या विधवारूपधारिणी

वगलाया दक्षभागे एकवक्त्रं प्रपूजयेत्

महारुद्रेति विख्यातं जगत्संहारकारकम्

***मातंगीदक्षिणांशे च मतंगं पूजयेच्छिवम्\|\|***

  

where Matangi is considered as the Shakti of Shiva, Matanga.

  

  

For Matanga's legend,

  

13,003।019a स्थाने मतंगो ब्राह्मण्यं नालभद् भरतर्षभ  
13,003।019c चण्डालयोनौ जातो हि कथं ब्राह्मण्यम् आप्नुयात्\|  

  

Mahabharata provides some clue. 003-028. How he became Brahmana sage.

  

Brahmanda Purana provides the birth of Matangakanya:

  

maata"ngasya mahar.sestu tasya svapne tadaa mudaa /

taapicchama\~njariimekaa.m dadau kar.naavata.msata.h // BndP_3,31.102 //

tatsvapnasya [prabhaave.na](http://prabhaave.na) maata"ngasya sadharmi.nii /

**naamnaa siddhimatii garbhe laghu"syaamaamadhaarayat // BndP_3,31.103 //**

**tata eva samutpannaa maata"ngii tena kiirtitaa /**

**laghu"syaameti saa prokta "syaamaa yanmuulakandabhuu.h // BndP_3,31.104 //**

maata"ngakanyakaa h.rdyaa.h ko.tiinaamapi ko.ti"sa.h /

laghu"syaamaa mahaa"syaamaamaata"ngii v.rndasa.myutaa.h /

a"nga"saktitvamaapannaa.h sevante priyakapriyaam // BndP_3,31.105 //

iti maata"ngakanyaanaamutpatti.h ku.mbhasa.mbhava /

kathitaa.h saptakak.saa"sca "saalaa lohaadinirmitaa.h // BndP_3,31.106 //

  

More may be found on search. So Matanga, a candala, became Muni and after he became Muni, Matangi was born, to reconcile two versions.
