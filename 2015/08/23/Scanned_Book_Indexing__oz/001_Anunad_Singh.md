Suppose the font used in pdf Devanagari is Krutidev and you want to search सीताराम in it. Convert सीताराम (which is in Unicode) to Krutidev using any font converter. Search the resulting text (lhrkjke)
in the pdf file and you will get appropriate results.

  

I use it quite frequently.

  

But those PDF files whichuse Unicode Devanagari have many problems. For example, You will not get the same text as seen on the screen if you select and copy-paste in some text box. Similarily, you may not find a word when you search in such PDF files even though you see the wordin front of you.
